

### get started

```bash
 $ sudo npm install hbuild-cli -g
```

### Usage

``` bash
$ h init new-project ;
```

### Local Templates

when you clone this project,you can  use a template on your local file system:

```bash
$ git clone git@github.com:hawx1993/hbuild.git
$ h init ./hbuild new-project
```
